# Labelpilot.de — Keyword Map Germany

## 1. Purpose

This document defines the Germany-focused keyword map for **Labelpilot.de**.

Labelpilot.de is a German-language, Germany-focused, B2B-first custom PP roll label ordering and reorder platform.

The keyword map must guide:

- SEO page planning
- programmatic SEO
- metadata
- internal linking
- content priorities
- Google Ads later
- GEO / AI-search content
- conversion-focused landing pages

This is not a generic print keyword list.

This keyword map focuses on German B2B buyers who need recurring product labels.

---

## 2. Strategic Keyword Verdict

The correct keyword strategy is:

> Target German commercial and B2B-intent keywords around PP roll labels, product labels, food labels, supplement labels, beverage labels, transparent/opake labels, 100×200 mm labels, file upload and reorder.

The wrong keyword strategy is:

> Chase broad generic keywords like “Aufkleber”, “Druckerei”, “Flyer drucken”, “Visitenkarten” or consumer sticker traffic.

Labelpilot.de should win qualified B2B intent, not irrelevant traffic.

---

## 2a. Visual-First SEO Rule (Mandatory)

Keyword strategy defines **intent and targeting**, not page format.

Every keyword/SEO page must present its content **visually**, not as a text wall.

Required structure on each keyword page:

```txt
short direct-answer block (1-2 short German sentences)
product spec table (Material, Größe, Auflage, Lieferform)
comparison table where intent is comparison (PP vs Papier, transparent vs opak)
FAQ accordion (concise German questions and answers)
visual proof (Rollenetiketten-, Material- oder Packaging-Mockups)
clear German CTA (Jetzt konfigurieren, Musterbox anfordern, Angebot anfordern)
internal links to primary, reorder and quote pages
```

Hard limits:

- No paragraph longer than 2 desktop lines.
- No repeated keyword stuffing to hit a density target.
- No thin or duplicated AI-generated pages spun up only for a keyword variant.
- One keyword cluster = one substantive page; variants are covered inside that page, not as near-duplicate pages.
- Indexable German content stays, but it is delivered through tables, accordions and answer blocks — never long essays.

A page may target high intent and still fail if it reads as an SEO text wall. Visual-first delivery is a launch requirement, not optional polish.

## 4. Keyword Intent Types

Use these intent labels:

```txt
COMMERCIAL_HIGH
COMMERCIAL_MEDIUM
INFORMATIONAL_COMMERCIAL
INFORMATIONAL_GENERAL
BRAND
REORDER
QUOTE
```

### 4.1 COMMERCIAL_HIGH

Buyer is close to ordering or requesting quote.

Example:

```txt
supplement etiketten drucken
lebensmitteletiketten drucken
pp rollenetiketten drucken
```

### 4.2 COMMERCIAL_MEDIUM

Buyer is comparing materials or options.

Example:

```txt
transparente pp etiketten
opake pp etiketten
rollenetiketten 100x200
```

### 4.3 INFORMATIONAL_COMMERCIAL

Buyer is researching but may convert.

Example:

```txt
pp etiketten vs papieretiketten
druckdaten etiketten vorbereiten
```

### 4.4 INFORMATIONAL_GENERAL

Mostly educational. Lower priority.

### 4.5 BRAND

Brand search for Labelpilot.de.

### 4.6 REORDER

Repeat-order search.

### 4.7 QUOTE

B2B offer request intent.

## 23. Final Verdict

The correct keyword strategy for Labelpilot.de is:

> German B2B commercial intent around PP roll labels, food labels, supplement labels, beverage labels, 100×200 mm labels, reorder and quote requests.
