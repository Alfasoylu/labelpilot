## 4. Phase Goal

The Phase 2 goal is:

> strengthen P0 commercial pages and add only a small set of high-quality P1 and guide pages with correct metadata, schema, canonicals, sitemap rules, and internal links.

---

## 5. In Scope

Phase 2 includes:

1. P0 metadata completion
2. Canonical verification
3. Sitemap and robots alignment
4. Visible FAQ sections where FAQ schema is used
5. BreadcrumbList implementation
6. Product, Service, Organization, and WebSite schema alignment
7. Direct answer blocks
8. Internal linking completion
9. Up to four guide pages if quality is high
10. Up to four P1 commercial pages if quality is high

Phase 2 excludes:

1. Mass programmatic SEO
2. City pages
3. Thin glossary or guide pages
4. Template-library SEO rollout
5. Generic print topics

## 9. Technical SEO Requirements

```txt

```

```txt

```

---

## 10. Content Structure Requirements

Each indexable Phase 2 page must include:

1. German H1
2. Kurzantwort near the top
3. Specific B2B buyer context
4. Structured content block or table
5. CTA
6. Internal links
7. Metadata
8. Correct canonical

If FAQ schema is used, FAQ content must be visible on-page.

---

## 11. Sitemap Gate

Do not add a page to sitemap until it has:

```txt
metadata
canonical
internal links
useful German content
clear buyer intent
CTA
no duplicate or thin content
```

If a page does not pass:

```txt
noindex
not in sitemap
```
