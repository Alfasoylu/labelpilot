# Labelpilot.de — SEO Strategy 2026

## 1. Customer-Facing Language Rule

The Labelpilot.de MVP is **German-language only** on all public customer-facing pages.

This is a hard SEO rule.

Primary SEO language:

```txt
German
```

Secondary future language:

```txt
English may be added later, but not in MVP.
```

Turkish is internal only.

Do not create English public pages during MVP.

---

## 2. Strategic SEO Verdict

Labelpilot.de must not compete as a generic online print shop.

The correct SEO position is:

> B2B custom PP roll labels for German food, beverage and supplement brands.

German positioning:

```txt
Individuell bedruckte PP-Rollenetiketten für Lebensmittel-, Getränke- und Supplement-Marken in Deutschland.
```

The wrong SEO position is:

> Cheap stickers, business cards, flyers and generic printing.

---

## 3. MVP SEO URL Structure

All public MVP URLs use German slugs under `/de`.

Required P0 pages:

```txt
/de
/de/lebensmittel-etiketten
/de/supplement-etiketten
/de/getraenke-etiketten
/de/transparente-pp-etiketten
/de/opake-pp-etiketten
/de/pp-rollenetiketten
/de/etiketten-100x200
/de/angebot-anfordern
/de/nachbestellen
```

No English versions in MVP.

Do not create:

```txt
/en
/en/food-labels
/en/supplement-labels
/en/custom-labels
```

---

## 4. German Metadata Rule

Every public page must define German:

- title
- description
- H1
- CTA
- FAQ
- schema-supported visible content

Example title:

```txt
Supplement-Etiketten drucken | Labelpilot.de
```

Example description:

```txt
Bedruckte PP-Rollenetiketten für Supplement-Marken in Deutschland. 100×200 mm, opak oder transparent, mit gespeicherter Druckdatei und einfacher Nachbestellung.
```

---

## 5. German CTA Rule

Allowed CTAs:

```txt
Jetzt konfigurieren
Angebot anfordern
Musterbox anfordern
Etiketten nachbestellen
Druckdatei hochladen
Druckdatei später senden
Proof freigeben
```

Not allowed in MVP:

```txt
Configure now
Request quote
Order sample
Reorder labels
Upload artwork
Send file later
Approve proof
```

---

## 6. Compliance Disclaimer

Food, beverage and supplement pages must show this German disclaimer:

```txt
Hinweis: Für rechtliche Pflichtangaben, Zutaten, Nährwerte, Allergene, Health Claims und regulatorische Inhalte ist der Kunde verantwortlich. Labelpilot.de übernimmt Druckproduktion, technische Dateiprüfung und Layout-Unterstützung, jedoch keine rechtliche Prüfung.
```

---

## 7. Final SEO Rule

Codex must build German-first SEO pages only for MVP.

If English pages are added later, hreflang, separate metadata, translation QA and content strategy must be documented first.
